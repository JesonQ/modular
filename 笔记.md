## ES6的总结


* 三种暴露方式
* 分别暴露   export let a = 100; export function foo(){};
* 统一暴露   export {foo, name}
* 默认暴露   export default {}  可以暴露任意JS数据类型 一般是暴露对象

* 三种导入方式
* 通用导入 import * as m2 from "./m2"
* 解构导入 import {name, foo} from "./m1"
* 简便导入 import m3 from "./m3"  注意:只适用于默认暴露
